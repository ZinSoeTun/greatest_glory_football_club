Local Demo

Public visitor တွေက ဒီ browser ထဲက settings ကို မသုံးပါဘူး။ လူတိုင်း Google Drive data မြင်ရအောင် `assets/scripts/config.js` ထဲက `publicApiUrl` မှန်နေပြီး GitHub Pages site ကို redeploy လုပ်ထားဖို့ လိုပါတယ်။

Admin feature တွေကိုသုံးဖို့ အတည်ပြုဝင်ရန် လိုအပ်ပါတယ်။
Settings ကို သိမ်းပြီးရင် local demo mode (သို့) Google Drive mode နဲ့ ချိတ်ထားတာကို စမ်းကြည့်နိုင်ပါတယ်။

Local demo mode က browser ထဲမှာပဲ data သိမ်းပြီး demo / design preview အတွက် သင့်တော်ပါတယ်။ Google Drive mode ကတော့ Apps Script session login သုံးပြီး CSV file နဲ့ uploaded media တွေကို online sync လုပ်ပေးပါတယ်။
